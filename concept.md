# 企画書：キーボード完結型カーソルジャンプツール

## 1. 企画概要

本企画は、Windows環境においてマウス操作を必要とするGUI操作を、キーボードのみで高速に実行できるようにする常駐型アプリケーションである。

大きなディスプレイやマルチモニター環境では、マウスカーソルを目的の位置まで移動するために手をホームポジションから離す必要があり、作業効率や集中力の低下につながる。

本アプリでは、画面を碁盤の目のようなグリッドに分割し、キー入力によって目的位置付近へカーソルを瞬時に移動させる。その後、必要に応じてキーボードによる微調整を行い、クリック・右クリック・ドラッグなどの操作を実行する。

これにより、マウスに手を移動することなく、GUI操作を素早く行える環境を実現する。

---

## 2. 解決したい課題

Windows環境では、ショートカットキーだけでは操作できず、GUIからしか操作できない機能が存在する。

たとえば以下のような操作である。

- ボタンのクリック
- チェックボックスの選択
- タブの切り替え
- コンテキストメニューの操作
- 業務アプリケーション上の固定位置ボタン操作
- レガシーWindowsアプリケーションのGUI操作

自作キーボードやキーカスタマイズによって、カーソル移動をキーに割り当てることは可能である。しかし、大きなディスプレイではキーによる連続的なカーソル移動だけでは移動距離が大きく、操作に時間がかかる。

そのため、以下の課題がある。

- マウスに手を移す動作が面倒
- ホームポジションが崩れる
- 大画面ではカーソル移動距離が長い
- キーボード中心の作業中にGUI操作が挟まると集中が途切れる
- Windowsの業務アプリや開発ツールでは完全なキーボード操作が難しい

---

## 3. 提案する解決策

画面全体をグリッド状に分割し、キー操作によって目的地点付近までカーソルを瞬間移動できる仕組みを提供する。

基本操作は以下の流れとする。

1. ホットキーでグリッドモードを起動する
2. 画面上にグリッドを表示する
3. キー入力で目的エリアを選択する
4. 必要に応じて選択エリアをさらに細分化する
5. 選択した位置へカーソルを瞬間移動する
6. キー操作で微調整する
7. キー操作でクリック・右クリック・ドラッグを行う

この方式により、マウスを使わずに画面上の任意の位置へ素早くアクセスできる。

---

## 4. コンセプト

### コンセプト名

**GridMouse**

### 一言で表すと

> Windows大画面ユーザーのための、キーボード完結型カーソルジャンプツール

### 提供価値

マウス操作を完全に置き換えるのではなく、  
**マウスに手を移す回数を減らし、キーボード中心の作業を中断させないこと**を目的とする。

## 7. MVP仕様

### 7.1 基本機能

| 機能 | 内容 |
|---|---|
| 常駐アプリ | Windows起動中にバックグラウンドで常駐する |
| グローバルホットキー | 任意のアプリ上でグリッドモードを起動できる |
| グリッド表示 | 画面を3×3または4×4に分割して表示する |
| セル選択 | キー入力によってグリッド内のセルを選択する |
| 段階的絞り込み | 選択セルをさらに細分化して目的位置に近づける |
| カーソル瞬間移動 | 選択したセルの中央へカーソルを移動する |
| 微調整 | キー操作で上下左右にカーソルを少しずつ移動する |
| クリック操作 | 左クリック、右クリック、ダブルクリックを実行する |
| キャンセル | Escキーでグリッドモードを解除する |
